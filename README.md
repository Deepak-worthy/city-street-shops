# city-street-shops

<h1>An E-commerce Web Application which helps street shops to sell their stuff
online to customers within the same city.</h1>

<h2>Tech Stack: </h2><br>
Reactjs<br>
Redux<br>
Express.js<br>
Nodejs<br>
MongoDB<br>
Mongoose<br>

<h2>Tools</h2>
<ul>
  <li>Multer for uploading files</li>
  <li>AWS S3 bucket for media file Storage</li>
  <li>MongoDB Atlas cloud database</li>
  <li>Hosted on Heroku</li>
</ul>
<h2>Features</h2>
<ul>
  <li>JWT Authentication for login</li> 
  <li>Feature to send order email to both customer and seller using nodemailer.</li>
  <li>Google Maps API for selecting delivery address</li>
  <li>Implemented Advanced feature for customer to chat with admin using socket.io </li>
  <li>Feature for online order payment implemented using Razorpay payment API.</li>
</ul>
